# 📝 Remember — Tasks & Notes

A simple, mobile-friendly web app to remember **tasks and notes**, with instant **search** and **custom tags**.

![status](https://img.shields.io/badge/status-ready-brightgreen)

## ✨ Features
- ✅ **Tasks** with checkboxes (mark as done)
- 📄 **Notes** for free-text reminders
- 🔍 **Instant search** — type any word, list filters live
- 🏷️ **Custom tags** — type your own (e.g. `work`, `urgent`, `shopping`) or pick from suggestions
- 💾 **Auto-save** in your browser (localStorage) — works offline
- 📱 **Mobile-friendly** — works great on phone and desktop

## 🚀 How to use
1. Open `index.html` in any browser, **or** visit the live site (GitHub Pages link below).
2. Type a task/note, pick or type a tag, choose Task/Note, click **Add**.
3. Use the search box to find items instantly.
4. Click tag chips to filter by category.
